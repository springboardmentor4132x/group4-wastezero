# Milestone 3 – Matching & Communication

## Objective

The goal of this milestone was to implement intelligent matching between volunteers and opportunities along with real-time communication.

## Features Implemented

* Volunteer-Opportunity Matching System
* Opportunity Recommendations based on location and skills
* Application Workflow Integration
* Real-time Messaging System
* Notification System for updates
* Chat Interface for communication

## Technologies Used

* Angular
* Node.js & Express
* MongoDB
* WebSockets (Socket.IO)

## Outcome

The platform now supports real-time interaction and intelligent matching, improving user engagement and usability.
