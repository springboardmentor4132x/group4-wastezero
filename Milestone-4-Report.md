# Milestone 4 – Admin Dashboard, Reporting & Testing

## Objective

The objective of this milestone was to implement administrative controls, reporting features, and complete system testing.

## Features Implemented

* Admin Dashboard with system statistics
* User and Opportunity Monitoring
* Application Tracking
* Report Generation (Users, Opportunities, Applications)
* Administrative Controls (block/remove users)
* System Testing and Validation

## Technologies Used

* Angular
* Node.js & Express
* MongoDB

## Outcome

The system became fully functional with monitoring, reporting, and testing completed, making it ready for real-world usage.
